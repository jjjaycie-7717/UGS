# UGS
用户增长系统控制台静态页，提供合作方、会员、后台入口。
依赖：Tailwind CSS CDN、Font Awesome CDN，部分页面用 Chart.js CDN。
本地预览：浏览器直接打开`index.html`，或 `npx serve .`。

## 整体逻辑
- 入口：登录后在 `index.html` 选择分销、会员或管理后台。
- 分销链路：合作方在 `partner-dashboard` 获取邀请码/链接→分销用户→`commission-list` 生成待结算分润→结算入账后在 `partner-wallet` 可提现。
- 会员链路：用户在 `user-center` 查看会员状态、续费/升级，订单详情在 `order-list`，邀请好友引导至 `invite` 落地页，生成新增邀请记录与统计。
- 管理链路：管理员在 `admin-partners` 审核合作方、调整佣金；财务数据在 `admin-finance` 对账，展示营收/佣金/退款与渠道占比。

## 角色视角
- 合作方：看板（数据与趋势）、钱包提现、佣金流水，推广物料为邀请码/链接/二维码。
- 用户会员：查看会员到期、续费或升级；复制邀请链接/码；查看订单与开票状态；落地页承担扫码登录验证与复制引导。
- 管理员：合作方列表审核（单选/全选批量通过）、佣金比例设置；财务对账查看营收、佣金支出、退款扣回、渠道占比，并核对明细。

## 页面
- `index.html`：主入口三卡片（分销、会员、后台）。
- `partner-dashboard.html`：合作方看板，四项指标卡；邀请码/链接/二维码展示与复制下载；近7天邀请/转化折线图（Chart.js）；最近邀请表；导航到钱包、佣金流水。
- `partner-wallet.html`：可提现/在途/累计卡片；提现记录表；提现表单（金额、账户、备注）与周期提示。
- `commission-list.html`：分润结算概览卡片（待结算/已入账/失效）；佣金流水表；状态筛选。
- `user-center.html`：会员状态与续费/升级按钮；邀请链接与邀请码复制；邀请统计三卡片；订单表（支付状态、开票状态）；复制脚本。
- `order-list.html`：订单列表页，搜索框+状态筛选，下方订单表。
- `invite.html`：邀请落地页，Tailwind 自定义主题；复制邀请需先微信扫码登录（模拟）；邀请统计三卡片；邀请记录动态渲染；二维码弹窗、登录轮询、复制提示。
- `admin-partners.html`：后台合作方列表，搜索与排序；待审核/总数/转化卡片；审核弹窗（全选、批量通过）；佣金设置弹窗；导航到财务。
- `admin-finance.html`：财务对账；营收/佣金/退款/净收益卡片；营收与佣金折线图、渠道占比饼图（Chart.js）；日期筛选；对账明细表。
- `白色logo.png`：页面 logo。

## 业务流（静态示意）
- 登录后进入 `index.html` 选择入口。
- 合作方：`partner-dashboard` 查看数据→跳转钱包提现或佣金流水；邀请码/链接推广；Chart.js 展示趋势。
- 合作方提现：在 `partner-wallet` 填写金额与账户提交；记录表显示历史，提示 T+7。
- 分润结算：`commission-list` 查看待结算/入账/失效，流水表列出佣金与结算时间。
- 用户侧：`user-center` 查看会员状态、续费/升级、复制邀请；查看订单；统计邀请效果。
- 订单列表：`order-list` 聚合支付记录，支持筛选。
- 邀请落地：`invite` 页面引导复制链接（需模拟扫码登录），统计展示并动态追加邀请记录。
- 管理后台：`admin-partners` 管合作方，审核弹窗批量通过，调整佣金比例；导航到财务。
- 财务对账：`admin-finance` 查看收入、佣金、退款、净收益，Chart.js 展示趋势与占比，明细表核对。
